# Airlines Price Prediction Analysis (Power BI)

This project analyzes airline ticket price variations using Power BI.

## Dashboard Insights
- Price variation across airlines
- Effect of last-minute booking
- Price variation based on departure and arrival time
- Source and destination price comparison
- Economy vs Business class price differences

## Tools Used
- Power BI
- Excel Dataset
- Data Visualization
